# 根付き木 [Root Tree]
木構造の一種で、特定の「根」と言われるノードが存在する。
この「根」を起点として、他の全てノードが人方向に配置される階層構造。

## 基本構造
- 根 Root
- 節 Node
- 葉 Leaf
- 内部節 Internal Node
- 親 Parent
- 兄弟 Sibling
- 深さ Depth
- 高さ Height
    特定のノードから最も遠いはまでのエッジの数
- 次数 Degree
    ノードが持つ子の数
